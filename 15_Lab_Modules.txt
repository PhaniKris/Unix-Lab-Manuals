8. Check the python version now

9. To support web application deployed on this system, install postgresql version 9.6
10. Check the version of postgresql by the command ==> postgres --version
11. Now remove postgresql 9.6 version from the system.
12. Now, install an upgraded version of postgresql with version 10 and check the version after installation
